How to Get Admin JWT Token for Shop Assistant

This guide explains how to obtain an admin JWT token from auth-microservice for use in the Shop Assistant admin panel.

Prerequisites

  • You have access to the auth-microservice (production: https://auth.statex.cz/admin or local dev)
  • You have a user account in the auth-microservice (or can create one)

Method 1: Using the Auth Admin Panel (Easiest)

Step 1: Open the Auth Admin Panel

  • Production: Open https://auth.statex.cz/admin
  • Local dev: Open http://localhost:3372/admin (if running frontend locally)

Step 2: Log In

  1. Enter your email and password
  2. Click the login button
  3. After successful login, you'll see the admin dashboard

Step 3: Get the JWT Token via Browser DevTools

  1. Open Developer Tools (F12 or right-click → Inspect)
  2. Go to the Network tab
  3. Look for the POST /auth/login request
  4. Click on it to view details
  5. Go to the Response tab
  6. Find the JSON response with accessToken: 
    {
  "user": { ... },
  "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "refreshToken": "..."
}
  7. Copy the entire accessToken value (it's a long string starting with eyJ...)

Step 4: Use the Token in Shop Assistant Admin

  1. Go to Shop Assistant Admin Panel
  2. Paste the accessToken into the "JWT token" field
  3. Click "Save token"
  4. You should see "Token saved" confirmation

Method 2: Using API Call (curl/Postman)

Step 1: Make a Login Request

Production:

curl -X POST https://auth.statex.cz/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your-email@example.com",
    "password": "your-password"
  }'

Local dev:

curl -X POST http://localhost:3370/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your-email@example.com",
    "password": "your-password"
  }'

Step 2: Extract the Token

The response will be JSON:

{
  "user": { ... },
  "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "refreshToken": "..."
}

Copy the accessToken value and paste it into the Shop Assistant admin panel.

Method 3: Using the Helper Script

If you have SSH access to the production server:

  1. SSH to the server: ssh statex
  2. Navigate to the shop-assistant directory
  3. Run the helper script: 
    ./scripts/get-admin-jwt.sh https://auth.statex.cz
  4. The script will output the JWT token - copy it and paste into the admin panel
Note: The script requires TEST_EMAIL and TEST_PASSWORD to be set in auth-microservice/.env.

Creating a Test User (If You Don't Have One)

If you need to create a test user:

  1. Set TEST_EMAIL and TEST_PASSWORD in auth-microservice/.env
  2. Run the create-user script: 
    cd auth-microservice
./scripts/create-test-user.sh
  3. Then use Method 1 or 2 above to get the JWT token

Troubleshooting

"Invalid credentials" error

  • Double-check your email and password
  • Ensure the user account exists and is active
  • Try resetting your password via the auth admin panel

"Invalid or missing JWT" error in Shop Assistant

  • Make sure you copied the entire accessToken (it's a long string)
  • Ensure there are no extra spaces before/after the token
  • Try getting a fresh token (tokens expire after 7 days by default)

Token expired

  • Get a new token using any of the methods above
  • Tokens can be refreshed using the refreshToken via POST /auth/refresh
